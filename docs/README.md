# docs フォルダ

このフォルダには、README やリポジトリ紹介で使う資料・画像を置きます。

## 必要なスクリーンショット（撮影後にここへ配置）

ルートの README.md は、以下のファイル名で画像を参照しています。同じ名前で保存してください。

| ファイル名 | 内容 | 撮り方の目安 |
|---|---|---|
| `screenshot-builder.png` | ビルダー画面（作る側） | ビルダーの「2.データ・画像」など、編集中の画面全体。横長（16:9 程度）が見栄え良い |
| `screenshot-viewer.png`  | ビューワー画面（見る側） | 公開された地図を実際に表示し、ピンやモーダルが見える状態 |

### 撮影のコツ
- ブラウザの開発者ツールやブックマークバーは隠してから撮ると、すっきり見えます。
- 個人情報（実名・連絡先など）が写り込まないサンプルデータで撮影してください。
- 横幅 1280〜1600px 程度の PNG が、README 上で鮮明かつ軽量です。
