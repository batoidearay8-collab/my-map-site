# 変更履歴 (Changelog)

本プロジェクトのバージョンごとの変更点を記録します。
形式は [Keep a Changelog](https://keepachangelog.com/ja/1.0.0/) に概ね準拠し、[セマンティック バージョニング](https://semver.org/lang/ja/) を採用しています。

## [10.19.0] - 2026-06

### 修正 (Fixed)
- **テンプレート切り替えで屋内/屋外が変わらない不具合を本当に修正**。テンプレート定義が2か所（使われていない `StepTemplate.tsx` と、実際に使われている `BuilderPage.tsx`）に存在し、これまで前者だけを修正していたため反映されていなかった。実際に使われている `BuilderPage.tsx` 側の `applyTemplate` を修正し、テンプレートに応じて屋内/屋外モードが切り替わるようにした。
- **文化祭テンプレートのおすすめカテゴリを空に修正**。`BuilderPage.tsx` 側の文化祭テンプレートに残っていた `["屋内マップ", "複数フロア", "QRコード配布", "入退口表示"]` を空に変更。

### 変更 (Changed)
- 全テンプレートに屋内/屋外の別を定義（観光・ライブ・便利・祭り・アウトドア＝屋外、文化祭・防災＝屋内）。
- テンプレートのカードと選択時のプレビューに、屋内/屋外の表示を追加。

## [10.18.0] - 2026-06

### 変更 (Changed)
- **「CSVに反映」ボタンを画面上部の固定「保存」ボタンに一本化**。あちこちに散らばっていて分かりにくかった保存操作を、画面上部に常時表示される1つの保存ボタンに統合しました。
  - ステップ2（データ・画像）の反映ボタンを削除。代わりに「画面上部の保存ボタンで保存できます」という案内を表示。
  - ステップ3（できあがり確認）の反映ボタンを削除。
  - 上部の保存状態バッジをクリック可能な保存ボタンに変更。未保存の変更があるときは赤く強調＋点滅で「💾 保存する」、保存済みのときは「✅ 保存済み」を表示。これにより、保存し忘れて次に進んでしまう問題を防止。
  - ※ CSVテキストを直接手書き編集する上級者モードの取り込みボタンは、役割が異なるため残しています。

## [10.17.0] - 2026-06

### 修正 (Fixed)
- **テンプレートを変更しても屋内／屋外が切り替わらない不具合を修正**。テンプレート選択のボタンが未定義の関数を呼んでいたため、何も反映されていなかった問題を解消。

### 変更 (Changed)
- **テンプレート選択を確認フロー化**。テンプレートのカードを押すと、即時反映ではなく「何が変わるか」をプレビュー表示するようになった。
  - 表示モード（屋内⇄屋外）の変更を before → after で明示。
  - 追加されるカテゴリ（アイコン付き）とサンプル地点の一覧を表示。
  - サンプルデータを入れるかどうかをチェックボックスで選択可能。
  - 「このテンプレートにする」を押すと、短いロード表示のあと反映される。
  - 各カードに屋内／屋外の別と「適用中」表示を追加。
- 文化祭テンプレートのおすすめ表示（reco.needs）は空のまま（要望どおり変更なし）。

## [10.16.0] - 2026-06

### 追加 (Added)
- **フロア作成（FloorPlanEditor）第2段階**。
  - **記号の配置**: 階段・トイレ・出入口・EV・AED・案内の6種類の記号を、パレットから選んでキャンバスに配置できる。記号はドラッグで移動、ダブルクリックで削除。書き出しPNGにも反映。
  - **ラベル文字サイズの自動調整**: 部屋の大きさに合わせてラベルのフォントサイズを自動計算。小さい部屋でも文字がはみ出さず、大きい部屋では見やすく表示。プレビューと書き出しの両方に適用。
  - **元に戻す（アンドゥ）**: 部屋の追加・移動・拡大縮小・回転・色変更・削除・レイヤー変更・全クリアを1手ずつ取り消せる。「元に戻す」ボタンと Ctrl/Cmd+Z に対応（履歴は最大50手）。

## [10.15.0] - 2026-06

### 追加 (Added)
- **フロア作成（FloorPlanEditor）の部屋編集機能**。描いた後の部屋を自由に編集できるようになりました。
  - **移動**: 部屋をドラッグして平行移動（グリッド吸着対応）。
  - **拡大縮小**: 選択中の部屋の四隅ハンドルをドラッグ、またはサイドパネルの幅・高さ数値入力でリサイズ。
  - **角度の変更**: 部屋の上部の回転ハンドル、またはサイドパネルのスライダーで中心まわりに回転（吸着ONで15°刻み）。画面プレビューと書き出しPNGの両方に反映。
  - **レイヤーの変更**: 重なり順を「最前面・最背面・前へ・後ろへ」で変更。部屋が重なったときの上下関係を制御できる。

## [10.14.0] - 2026-06

### 追加 (Added)
- **簡易フロア作成機能（FloorPlanEditor）の強化**。第1段階として、以下を追加。
  - **色分けプリセット**: 選択中の部屋に色を設定できる（標準・教室=オレンジ・水回り=青・特別室=緑・施設=グレーの5種）。書き出すPNGと画面プレビューの両方に反映。文化祭の見取り図のように、部屋の種類を色で区別できる。
  - **グリッド吸着（スナップ）**: 部屋を描くときに50px単位のグリッドに自動で揃う。チェックボックスでON/OFF切替可能。手で並べてもガタつかず、見取り図らしく整列する。

## [10.13.0] - 2026-06

### 追加 (Added)
- **簡易フロア作成機能（FloorPlanEditor）**。外部の画像編集ソフトを使わなくても、AtlasKobo内でフロア画像（間取り図）を作れるようになりました。
  - キャンバス上をドラッグして長方形（部屋）を追加し、名前を付けられます。
  - 「この画像を使う」を押すと、Canvas APIでPNGに書き出し、既存のフロア画像の仕組み（`setBuilderAsset` + `imageWidthPx/HeightPx`）にそのまま取り込みます。ビューワーや経路探索のコードは変更不要。
  - 単一フロア・複数フロアの両方に対応（各フロアの「🖊️描いて作る」ボタンから起動）。
  - これまで「複数階を作るには外部でフロア画像を用意する必要があった」制約を解消。

## [10.12.1] - 2026-06

### 修正 (Fixed)
- 左上のブランドロゴ画像（`brand/logo.png`）の黒い背景を透過化。ライトモードで黒い四角が目立っていた問題を解消。ビルダー側と公開サイト側（`site-template`）の両方を更新。

## [10.12.0] - 2026-06

### 追加 (Added)
- **テンプレート別サンプルデータの投入機能**。ビルダーの「テンプレート選択」画面で「サンプルを入れて始める」を押すと、選んだ用途（観光・文化祭・防災など全7種）に応じたサンプルのカテゴリと地点が自動で入り、白紙ではなく完成形から編集を始められます。
  - 屋外テンプレートは緯度経度、屋内テンプレート（文化祭・防災）は正規化座標で配置。
  - 既存の地点がある場合は確認ダイアログを表示してから置き換え。
  - サンプルのモードに合わせて屋内／屋外も自動で切り替え。

## [10.11.0] - 2026-06

### 追加 (Added)
- **GitHub 直接公開パネル** をビルダーの「4.公開」画面に統合。GitHub の Personal Access Token を入力するだけで GitHub Pages へ公開できるようになりました（事前の環境構築が不要）。
- MIT `LICENSE` ファイルを追加。
- `package.json` にライセンス・説明・著者・リポジトリ等のメタ情報を追加。
- 非技術者向けクイックスタートを README に追加。

### 変更 (Changed)
- ビルダー UI から研究モード（任意・学術利用の選択カード）およびマスターモードを削除し、公開向けに簡素化。

### 補足 (Notes)
- ビューワー側の研究用ログ機構（同意ダイアログ・匿名ログ）は残置していますが、**既定で無効（オプトイン方式）** です。配布される地図サイトでは利用者に表示されません。

## [10.4.0]

### 修正 (Fixed)
- `pickPoiName()` / `pickCategoryLabel()` の引数ミスマッチを修正。
- `DetailsModal` に正しい `mode` / `now` props を渡すよう修正。
- `QrModal` に `uiLang` prop を追加。
- `DropZone` の `label` → `title` prop を修正。
- `outdoor.centerLat/centerLng` → `outdoor.center[0]/[1]` を修正。

### 変更 (Changed)
- `ConfigSchema` に `reco` フィールドを正式追加。
- `template` enum を 7 種に拡張。
- `BuilderPage` 内の `as any` キャストを大幅削減。

### 追加 (Added)
- マーカークラスタリング（多数 POI の自動グループ化）。
- GeoJSON インポート / エクスポート（QGIS / uMap 連携）。
- 埋め込みコード生成（iframe 用、Step 4 でコピー）。
- ZIP 生成時のローディングインジケータ。

### UI/UX
- プライバシーバナーを dismiss 可能に。
- アクセシビリティ強化（スキップリンク、ARIA、キーボード操作、focus-visible）。
- ビルダーのモバイル レスポンシブ改善。
- 公開サイト CSS のライトモード配色をビルダーと統一。
